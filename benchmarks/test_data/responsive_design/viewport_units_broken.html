Broken Viewport Units

This layout demonstrates common viewport unit problems that break on mobile devices.

Why This Breaks

Mobile Safari

100vh doesn't account for the dynamic address bar, causing content to be cut off on mobile Safari.

Viewport Width Issues

Using vw for padding and text creates horizontal scrolling and unpredictable scaling across devices.

Poor Typography

Text sized with viewport units becomes unreadable on some screen sizes and creates accessibility issues.

Viewport Problems

This footer uses 100vh which compounds scrolling issues and creates an unnecessarily tall page on mobile devices.

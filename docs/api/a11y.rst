Accessibility (axe-core) API
=============================

Deterministic WCAG 2.1 A/AA accessibility checks via a vendored
`axe-core <https://github.com/dequelabs/axe-core>`_ bundle, injected into a
Playwright-rendered page. This engine requires no API key or LLM call.
